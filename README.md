slush-react
=============

React app generator used gulp

Usage
-----

1. 全局安裝 slush

  ```
  $ npm i -g slush
  ```

1. 刻隆本項目

  ```
  $ npm i -g @ersinfotech/slush-react
  ```

1. 在適合的位置，建立項目資料夾

  ```
  $ mkdir ~/awesome-react
  $ cd ~/awesome-react
  ```

1. 運行本項目生成器

  ```
  $ slush react
  ```

1. 回答問題，你已成功建好項目骨架

Upgrade
-------

  ```
  $ npm i -g @ersinfotech/slush-react@latest
  ```

Generators
----------

- **default**
  
  適用于項目初始化，建立項目骨架

  ```
  $ slush react
  ```
